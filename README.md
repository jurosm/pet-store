# Pet Store

SPA toy shop application for pets!

## Features
- .NET Core 3.1
- Angular  8.2.14
- PostgreSQL 12
- Docker 19.03.5
- Auth0
- Stripe testing 

## Environment variables

### .NET Core
- DB_CONNECTION_STRING
- AUTH0_DOMAIN
- AUTH0_AUDIENCE
- AUTH0_CLIENT_ID
- AUTH0_CLIENT_SECRET
- STRIPE_SECRET

### Angular
- API_URL
